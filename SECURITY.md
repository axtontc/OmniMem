# Security Policy

## Supported Versions

Only the latest stable release of OmniMem receives security updates.

| Version | Supported |
| --- | --- |
| 1.0.x | :white_check_mark: Yes |
| < 1.0.0 | :x: No |

## Reporting a Vulnerability

If you discover a security vulnerability within OmniMem, please report it immediately by opening an issue on GitHub with the `security` label, or by emailing the project maintainer directly at axtonc@gmail.com.

Please do not disclose the vulnerability publicly until a fix has been prepared and released. We aim to address all reported critical security issues within 48 hours.
